Etiquetas HTML

Etiqueta blockquote

La etiqueta HTML BLOCKQUOTE se utiliza para crear bloques de texto que representan el contenido citado de otra fuente. Sirve para indicar que el texto dentro de esta etiqueta es una cita, para que los navegadores y motores de búsqueda comprendan que se trata de contenido citado. Algunos de los usos y propósitos de la etiqueta BLOCKQUOTE en HTML son:

Destacar Citas:

La etiqueta BLOCKQUOTE se utiliza para destacar y formatear visualmente las citas dentro de un texto. Por defecto, los navegadores suelen mostrar el contenido citado con una sangría en los márgenes izquierdo y derecho.

Semántica:

Proporciona una estructura semántica para el contenido citado en una página web, lo que facilita la identificación de citas tanto para los navegadores como para los motores de búsqueda.

Accesibilidad:

Al marcar el contenido citado con la etiqueta BLOCKQUOTE, estás contribuyendo a hacer que tu página sea más accesible para personas con discapacidades visuales y ayudando a los lectores de pantalla a identificar las citas.

Ejemplo

El famoso físico Albert Einstein dijo una vez:

La vida es como andar en bicicleta. Para mantener el equilibrio, debes seguir adelante.

Esta es una sabia reflexión de Einstein sobre la vida y el progreso.

Etiqueta pre

La etiqueta pre en HTML se utiliza para definir texto preformateado. Esta etiqueta es especialmente útil cuando deseas mostrar texto que debe mantener su formato original, como código fuente, poesía, datos tabulares u otro contenido en el que la estructura y el espaciado son importantes.

Ejemplo

Las cosas que necesitarás están listadas aquí: - Una linterna. - Una caja de fósforos. - Un cuchillo afilado. 

Las cosas que necesitarás están listadas aquí:
              
                - Una linterna.
              
                - Una caja de fósforos.
              
                - Un cuchillo afilado.

Etiqueta figure-figcaption

La etiqueta FIGURE en HTML se utiliza para encapsular y representar contenido autónomo, como imágenes, gráficos, ilustraciones, diagramas, videos u otros elementos multimedia, junto con su respectiva descripción o leyenda. La etiqueta FIGURE proporciona un marco semántico para identificar y relacionar el contenido principal con su descripción o leyenda, que generalmente se agrega con la etiqueta FIGCAPTION El propósito principal de la etiqueta FIGURE es mejorar la accesibilidad y la estructura del contenido, ya que ayuda a los lectores de pantalla y a los motores de búsqueda a comprender la relación entre el contenido principal y su descripción.

Etiqueta main

Esta etiqueta se introdujo en HTML5 para mejorar la semántica y la accesibilidad de las páginas web al proporcionar una forma de identificar claramente cuál es el contenido central o principal de una página.

Etiqueta header

La etiqueta HEADER en HTML se utiliza para definir una sección de encabezado en una página web. Esta sección suele contener elementos que proporcionan información introductoria o de identificación sobre la página, como el logotipo del sitio, el título de la página, el menú de navegación principal u otros elementos de encabezado

Etiqueta section

La etiqueta SECTION en HTML se utiliza para dividir el contenido de una página web en secciones o bloques temáticos distintos y semánticamente significativos. Cada SECTION debe representar un tema o contenido autónomo dentro de la página. La etiqueta SECTION ayuda a organizar el contenido de manera estructurada y a mejorar la semántica de la página web.

Etiqueta em

La etiqueta EM en HTML se utiliza para enfatizar el texto dentro de un documento web. Generalmente, el texto enfatizado se muestra en cursiva en la mayoría de los navegadores web. La etiqueta EM se utiliza para dar énfasis a una parte del texto, lo que indica que esa parte es especialmente importante en el contexto del contenido.
